import

典型的一个原型链污染。
经过调试可知，正常情况下，就是让 obj 的对象增加一对 key:value
当第二个参数有。的时候，. 前面的内容记作 key1，. 后面的内容记作 key2。而后递归调用，
obj 对象中的 key1 对象增加一个键值对 key2:value。
如果说 key 替换成 proto.x1, 就会造成 obj 的__proto__的值发生改变。达成一个原型链污染。
